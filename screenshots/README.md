# 游戏截图说明

本文件夹用于存放游戏真实截图。

## 如何添加截图

1. 在浏览器中打开对应的游戏页面
2. 使用截图工具（如系统截图、Snipaste等）截取游戏界面
3. 保存为PNG格式，命名为：
   - `archery.png` - 射箭游戏
   - `ping-pong.png` - 乒乓球游戏
   - `basketball.png` - 篮球游戏
   - `soccer.png` - 足球游戏
   - `bowling.png` - 保龄球游戏
4. 将图片放入本文件夹
