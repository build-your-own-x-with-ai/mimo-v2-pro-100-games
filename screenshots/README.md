## 如何添加新截图

1. 在浏览器中打开对应的游戏页面
2. 使用截图工具（如系统截图、Snipaste等）截取游戏界面
3. 保存为PNG格式，建议命名为游戏相关名称
4. 将图片放入本文件夹
5. 更新主README.md中的对应链接
